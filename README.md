# Калькулятор стальных колонн промышленных зданий

Веб-калькулятор для подбора оптимального стального профиля колонн каркаса промышленных зданий по СП 16.13330 и СП 20.13330.

**Демо:** https://dist-ecuvkgbu.devinapps.com

## Что внутри

- 208 профилей (двутавры Б/Ш/К + кв./пр. трубы) × 4 марки стали (С255Б, С355Б, С245, С345) × 0–4 распорки = 2080 вариантов на расчёт
- Сортировка по полной массе колонны (профиль × высота + распорки × 12 кг × шаг × 1.15)
- Проверки по СП 16.13330: σ, σ уст X/Y, гибкость X/Y — все < 1.0
- Нагрузки по СП 20.13330: снег, ветер с k(ze), ζ(ze), ν(ρ,χ) для местности A/B/C, кровля, ограждения
- Автозаполнение w₀/Sg по 1096 населённым пунктам РФ
- Только СП — без СП РК EN

## Стек

- Vite + React + TypeScript
- Полностью клиентский, без бэкенда
- Деплой как статичный фронтенд

## Локальный запуск

```bash
npm install
npm run dev      # dev-сервер на localhost:5173
npm run build    # сборка в dist/
```

## Сверка с Excel

Расчётный движок воспроизводит формулы исходного Excel-калькулятора 1-в-1. Скрипты в `scripts/`:

- `verify_baseline.py` — проверка default-сценария (N=63.79 кН, M=143.90 кН·м)
- `excel_oracle.py` — пересчитывает Excel через LibreOffice для 10 разных сценариев и сохраняет ground-truth в JSON
- `compare_scenarios.py` — прогоняет те же входы через Python-перевод движка и сравнивает с эталоном (требует `openpyxl` и `libreoffice-calc-nogui`)

Все 10 сценариев совпадают с Excel с 0.00% расхождением по N и M.

## Структура

```
src/
  App.tsx                     UI (форма + таблица результатов)
  calc/
    engine.ts                 главный расчёт: подбор профиля
    wind.ts                   ветер по СП 20.13330 (k(ze), ζ(ze), ν)
    steel.ts                  Ry для марок стали С255Б/С355Б/С245/С345
    types.ts                  типы CalculationInput/Output
  data/
    profiles/profiles.json    208 профилей (A, Ix, Wx, ix, Iy, Wy, iy, mass)
    regions/                  база климатики (1096 НП)
  types/climate.ts            интерфейсы для климатических данных
```
