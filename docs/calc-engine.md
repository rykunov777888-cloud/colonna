# Ядро расчёта калькулятора стальных колонн

`src/calc/` (~500 строк TS) воспроизводит логику листа «Сводка» из исходного Excel:
считает усилия в основании одной колонны, перебирает все профили × марки стали ×
распорки, отсеивает не проходящие проверки и сортирует по полной массе.

Сверка с Excel: 10 сценариев, **0.00% расхождение** по N и M (см.
`scripts/excel_oracle.py` + `scripts/compare_scenarios.py`).

---

## 1. Вход (`types.ts`)

`CalculationInput` содержит:

- **геометрия:** пролёт, длина, высота, уклон кровли, шаг рам / шаг стоек
  фахверка, кол-во пролётов, тип кровли (двускатная / односкатная), наличие
  связей по периметру;
- **тип колонны:** `edge` / `middle` / `fachwerk`;
- **нагрузки:** w₀, Sg, q кровли, q ограждения, тип местности A / B / C, γₙ,
  надбавка %;
- **прочее:** μ (расч. длина), цены 4-х марок стали, кран (опционально).

---

## 2. Нагрузки от ветра (`wind.ts`, СП 20.13330)

Для высоты `h`:

1. **k(zₑ)** и **ζ(zₑ)** — линейная интерполяция по высоте отдельно для каждой
   местности (3 таблицы по 12 точек, 5 м – 350 м).

2. **ν(ρ, χ)** — коэффициент пространственной корреляции, билинейная
   интерполяция по таблице 7×7 (точки `[0.1, 5, 10, 20, 40, 80, 160] ×
   [5, 10, 20, 40, 80, 160, 350]`). Таблица и порядок интерполяции взяты 1-в-1
   из Excel:

   ```
   νLongB = ν(0.4·span, h)        // зона B при ветре поперёк длины
   νShortB = ν(span,    h)        // зона B при ветре поперёк пролёта
   νFGH+   = νLongB
   ```

3. Для каждой зоны:
   ```
   mean  = |w₀ · k · c · γf|
   pulse = mean · ζ · ν
   total = mean + pulse
   ```

4. На выходе:
   - **горизонт. давление** = `max(longB, shortB) + FGH+` (используется для
     момента),
   - **вертик. компонент** = `FGH+` (отсос с кровли).

С `γf = 1.4`. Затем результат умножается на γₙ.

---

## 3. Снеговая, кровля, стены (`engine.ts`)

- **Sснег** = `γf · Sg · cos(α) · γₙ` (γf = 1.4, α — уклон кровли).
- **q кровли**, **q ограждения** — введённые расчётные значения × γₙ.

---

## 4. Грузовая площадь и площадь стен

Зависят от типа колонны:

| тип       | tributary (вертикаль)             | wall (горизонт)        |
|-----------|-----------------------------------|------------------------|
| edge      | `(span / 2) · framePitch`         | `h · framePitch`       |
| middle    | `span · framePitch`               | `h · framePitch`       |
| fachwerk  | `(fachverkPitch · framePitch) / 2`| `h · fachverkPitch`    |

---

## 5. Усилия в основании

**N (осевая, кН):**

```
N = ((Sснег + Wверт + qкров) · Aверт + qогр · Aстен + Gкран) · (1 + надбавка/100)
```

**M (момент, кН·м):**

```
Mбаз = Wгор · pitch · h² / 2          // pitch = шаг рамы или фахверка
M    = Mбаз · k_момент + Mкран
```

`k_момент` — коэффициент перераспределения, табличный из Excel по типу колонны /
наличию связей / кол-ву пролётов:

| колонна   | связи  | один пролёт | многопролётная |
|-----------|--------|-------------|----------------|
| фахверковая | —    | 0.35        | 0.35           |
| крайняя   | нет    | 1.0         | 0.9            |
| крайняя   | есть   | 0.55        | 0.3            |
| средняя   | нет    | 1.0         | 0.6            |
| средняя   | есть   | 0.55        | 0.1            |

---

## 6. Перебор профилей (208 × марки × 0..4 распорки = до 2080 вариантов)

Для каждой комбинации:

### 6.1. Ry по `steel.ts`
С понижением 1 / 1.025 и шагом по толщине полки:
- **С255Б:** 255 → 245 → 235 → 235 → 225 → 215 (по толщинам ≤ 10/20/40/60/80/>80 мм);
- **С355Б:** 355 → 345 → 340 → 325 → 315 → 295 (≤ 20/40/60/80/100/>100 мм);
- **С245:** 240 (трубные);
- **С345:** 340 (трубные).

### 6.2. Гибкости (безразмерные)
```
λ̄x = (μ · H · 100 / ix) · √(Ry / E)
λ̄y = (H / (struts + 1) · 100 / iy) · √(Ry / E)   // распорка делит y-плоскость
```

### 6.3. φx, φy — коэффициент устойчивости
СП 16.13330, п. 7.1.3, с α/β по типу сечения (для двутавров α = 0.04,
β = 0.09 / 0.14):

```
δ = 9.87 (1 − α + β · λ̄) + λ̄²
φ = 0.5 · (δ − √(δ² − 39.48 · λ̄²)) / λ̄²
```

### 6.4. Пять проверок к-та исп. (γc = 0.95)

| проверка   | формула                                                                       |
|------------|-------------------------------------------------------------------------------|
| σ          | `(N / A + M / Wx) / (Ry · γc)`                                                |
| σ уст X    | `max(N / (A · φx / 2), M / (0.7 · Wx)) / (Ry · γc)`                           |
| σ уст Y    | `(N / (A · φy · c)) / (Ry · γc)`, где `c = min(1 / (1 + 0.7 · mx), 1)`,<br>`mx = (M / N) · (A / Wx)` |
| гибк. X    | `λx_actual / (180 − 60 · max(0.5, σx_compress))`                              |
| гибк. Y    | то же с `iy` и длиной `H / (struts + 1)`                                      |

Профиль **отсеивается**, если max(к-т исп.) > 1.

### 6.5. Масса и стоимость
```
масса колонны  = mass_per_m · H
масса распорок = struts · 12 кг · pitch · 1.15
общ. масса     = колонна + распорки
стоимость      = общ. масса · цена(сталь) / 1000   // т.р.
```

---

## 7. Сортировка и выдача

Все прошедшие комбинации сортируются по **возрастанию полной массы**, берутся
первые 50, выдаются в таблицу с пятью к-тами исп. и помечается лимитирующая
проверка (та, у которой к-т исп. максимальный).

В UI строки с **max(к-т исп.) > 0.95** подсвечиваются светло-красным —
индикатор работы профиля «на пределе».
